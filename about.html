About Me :: Keith Casko

Hey there, welcome to TaurusTech! I'm Keith, the person behind this little corner of the web. If you're wondering about the retro theme, well, I grew up with the dial-up sounds and pixelated screens of the 80s and 90s, and it definitely left an impression!

From Baker to Builder

My path into technology hasn't been exactly linear. For over 20 years, my world revolved around baking – precision, process, and ensuring everything came together just right. While I enjoyed that craft, a long-held fascination with technology kept calling.

I realized that many skills from baking – meticulous problem-solving, optimizing workflows, analytical thinking, and attention to detail – translate surprisingly well to the tech world. So, I decided to make the leap and pursue my passion for building and understanding systems.

Forging a New Path in Tech

Currently, I'm channeling that energy into my studies as a **Bachelor of Science in Software Engineering student at Western Governors University (WGU)**, aiming for graduation around 2029. It's an exciting journey covering everything from programming fundamentals to system administration.

My Tech Toolkit (Actively Expanding!)

I'm building skills across various domains:

  • Programming: Getting comfortable with Python for scripting and automation, and learning Bash for system tasks.
  • Systems & Networking: Working with Windows Server 2022 (including Active Directory!), Linux flavors like Ubuntu and Kali, and understanding network basics.
  • Tools & Concepts: Using VirtualBox and VMware for virtualization, Git for version control, VS Code, and diving into core software development concepts.

Hands-On: The Home Lab Experience

Theory is one thing, but I learn best by doing. I run a home lab environment where I can experiment and break things (and hopefully fix them!):

  • Setting up and managing a full Windows Server 2022 domain with Active Directory, users, and group policies.
  • Running a dual-boot system (Windows 11 / Ubuntu) and virtual machines (including Kali Linux for security practice).
  • Writing Python scripts for practical tasks like file organization.
  • Continuously troubleshooting network configurations and VM setups.

It's my sandbox for applying what I learn and exploring new technologies.

Future Focus: Cybersecurity

While I'm currently focused on software engineering, my long-term interest lies strongly in **cybersecurity**. Understanding how systems are built is crucial for learning how to protect them. My experience with server administration, networking, and different operating systems (especially Kali Linux in the lab) is all building towards that goal. I'm planning on tackling certifications like CompTIA A+ and the Google IT Support cert to solidify my foundational knowledge.

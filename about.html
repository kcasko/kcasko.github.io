About Me :: Keith Casko

Hey there, welcome to TaurusTech! I'm Keith, the person behind this little corner of the web. If you're wondering about the retro theme, I grew up with the dial-up sounds and pixelated screens of the 80s and 90s, and they definitely left an impression.

From Baker to Builder

My path into technology hasn’t been exactly linear. For over 20 years, my world revolved around baking: precision, process, and making sure everything came together just right. While I enjoyed that craft, a long-held fascination with technology kept calling.

I realized that many skills from baking, such as meticulous problem-solving, optimizing workflows, analytical thinking, and attention to detail, translate surprisingly well to the tech world. I decided to make the leap and pursue my passion for building and understanding systems.

Forging a New Path in Tech

Now I’m channeling that energy into my studies as a Bachelor of Science in Software Engineering student at Western Governors University (WGU), aiming for graduation around 2029. It’s been an exciting journey covering everything from programming fundamentals to system administration.

My Tech Toolkit (Actively Expanding)

I’m building skills across several domains:

  • Programming: Learning Python for scripting and automation, and Bash for system tasks.
  • Systems & Networking: Working with Windows Server 2022 (including Active Directory setup), Linux distributions like Ubuntu and Kali, and gaining a stronger understanding of network fundamentals.
  • Tools & Concepts: Using VirtualBox and VMware for virtualization, Git for version control, VS Code for development, and applying software engineering principles to real-world projects.

Hands-On: The Home Lab Experience

Theory is one thing, but I learn best by doing. I run a home lab environment where I can experiment, test, and troubleshoot:

  • Setting up and managing a full Windows Server 2022 domain with Active Directory, users, and group policies.
  • Running a dual-boot setup (Windows 11 and Ubuntu) with virtual machines for testing and security practice.
  • Writing Python scripts for practical automation and file management.
  • Constantly improving network configurations and refining system setups through hands-on troubleshooting.

It’s my sandbox for applying what I learn and exploring new technologies.

Future Focus: Cybersecurity

While I’m currently focused on software engineering, my long-term goal leans strongly toward cybersecurity. Understanding how systems are built is essential for learning how to protect them. My experience with server administration, networking, and different operating systems, especially Kali Linux in the lab, is all building toward that goal. I plan to pursue certifications like CompTIA A+ and Google IT Support to strengthen my foundational knowledge.
